# **App Name**: Sanjiwani Health

## Core Features:

- Doctor Discovery: Enable users to find nearby doctors and view their availability in real-time.
- Appointment Booking: Facilitate appointment booking for both in-person and virtual consultations.
- Diagnostic Booking: Search for diagnostic centers, view tests, and book appointments directly.
- Health Records: Provide secure storage for prescriptions and test reports.
- Hospital Search: Enable users to locate hospitals/clinics with details on facilities and services.
- Feedback System: Enable patient feedback for doctors, hospitals, and diagnostic centers.

## Style Guidelines:

- Primary color: A gentle blue (#72BCD4) to inspire trust and calm.
- Background color: Light grey (#F0F4F8) for a clean and modern interface.
- Accent color: A complementary teal (#4DB6AC) to highlight key actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a blend of modernity and warmth suitable for readability.
- Use clean and recognizable icons from a consistent set, indicating function and category
- Maintain a clean and modern layout with ample whitespace for readability.
